# Web Development Learning

Este repositório reúne meus estudos, exercícios e pequenos projetos desenvolvidos durante minha jornada de aprendizado em desenvolvimento web e programação.

O objetivo é registrar minha evolução, praticar conceitos fundamentais e construir uma base sólida antes de avançar para tecnologias mais complexas.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- Java
- PHP
- SQL

Outras tecnologias serão adicionadas conforme meus estudos avançarem.

## Estrutura

- HTML: Estrutura e semântica de páginas.
- CSS: Estilização, layouts e responsividade.
- JavaScript: Lógica de programação e manipulação do DOM.
- Git e GitHub: Controle de versão e colaboração.
- Java: Fundamentos da linguagem e orientação a objetos.
- PHP: Desenvolvimento back-end.
- SQL: Modelagem e consultas em banco de dados.
- Projetos: Aplicação prática dos conhecimentos adquiridos.

## Por que estudar o básico?

Mesmo com a evolução da Inteligência Artificial, dominar os fundamentos continua sendo essencial. Ferramentas de IA aceleram o desenvolvimento, mas compreender HTML, CSS e JavaScript permite analisar, corrigir, adaptar e manter o código com segurança e qualidade. Uma base sólida facilita o aprendizado de tecnologias mais avançadas e ajuda a tomar decisões técnicas de forma consciente.

## Objetivos

- Praticar constantemente.
- Aprender boas práticas de desenvolvimento.
- Construir uma base sólida em desenvolvimento web.
- Documentar minha evolução como desenvolvedora.
- Desenvolver projetos cada vez mais completos.
